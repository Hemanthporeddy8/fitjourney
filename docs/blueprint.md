# **App Name**: FitJourney

## Core Features:

- Photo Comparison: Allow users to upload daily photos and compare them, highlighting changes with a slider.
- AI Calorie Estimation: Use AI to estimate calories from food photos, providing a nutritional report. This feature will use a tool to determine if nutrients should be displayed to the user.
- Calorie Burner: Track walking/running distance and suggest simple exercises to burn estimated calories.

## Style Guidelines:

- Primary color: Teal (#008080) for a sense of health and vitality.
- Secondary color: Light gray (#D3D3D3) for a clean and modern look.
- Accent: Coral (#FF7F50) for call-to-action buttons and highlights.
- Clean and modern font for readability.
- Simple, outline-style icons for a minimalist aesthetic.
- Card-based layout for easy navigation and content organization.
